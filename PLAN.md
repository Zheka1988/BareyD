## 2. Дополнительно реализовано

- Поиск объекта по названию с центрированием
- Скрытие/показ объектов + панель скрытых + «Показать все»
- Счётчик объектов
- Инструменты измерения (расстояние, площадь, координаты)
- Панель рисования (карандаш, линия, круг, овал, прямоугольник, цвет, заливка, ластик)
- Экспорт в CSV / Excel
- Мини-карта
- Поиск внутри фильтров
- Привязка справочников к стране (модели + админка DAL)
- Валидация: обязательное имя, обязательная геометрия, проверка страны
- Страница входа (дизайн, фон с узлами и связями)
- Гибридная загрузка: серверная фильтрация и поиск вместо загрузки всех объектов
